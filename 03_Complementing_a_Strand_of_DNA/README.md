# Complementing a Strand of DNA (ID: REVC)

## مسئله


## ایدهٔ حل
- از متد `.replace()` پایتون برای هر نوکلئوتید استفاده می‌کنیم.
- پیچیدگی: O(n) که در آن n طول رشته است (هر count یک پیمایش انجام می‌دهد، ولی همچنان O(n) کلی).
- می‌توان با یک پیمایش و یک دیکشنری نیز شمارش کرد که برای رشته‌های خیلی بلند بهینه‌تر باشد.

## اجرا
```bash
python3 rna.py
```
و رشته زیر را به عنوان ورودی در خط فرمان وارد کنید:

GATGGAACTTGACTACGTAAATT
