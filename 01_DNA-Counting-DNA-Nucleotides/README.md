# Counting DNA Nucleotides (ID: DNA)

## مسئله
رشته‌ای از DNA شامل کاراکترهای A، C، G، T داده شده است. باید تعداد هر یک از این نوکلئوتیدها را شمرده و به‌ترتیب A, C, G, T چاپ کنیم.

## ایدهٔ حل
- از متد `.count()` پایتون برای هر نوکلئوتید استفاده می‌کنیم.
- پیچیدگی: O(n) که در آن n طول رشته است (هر count یک پیمایش انجام می‌دهد، ولی همچنان O(n) کلی).
- می‌توان با یک پیمایش و یک دیکشنری نیز شمارش کرد که برای رشته‌های خیلی بلند بهینه‌تر باشد.

## اجرا
```bash
python3 dna.py
```
و رشته زیر را به عنوان ورودی در خط فرمان وارد کنید:
